A `checkbox` is a graphical user interface element that allows users to select one or multiple options from a set. It displays a small square that can be checked or unchecked.

## Component Usage
### Default

A default `checkbox` is a form control that shows its initial state as either checked or unchecked when the page first renders, according to its initial configuration.

<SCheckbox client:only="vue" />
<Code code={SCheckboxRaw} lang="vue" title="default.vue" />

### Disabled

A disabled `checkbox` is a non-interactive form element, meaning users can't check or uncheck it. It's visually distinct, often dimmed, to signal that it's not available for selection.

<SCheckboxDisabled client:only="vue" />
<Code code={SCheckboxDisabledRaw} lang="vue" title="disabled.vue" />

### Required

A required `checkbox` is a form element that must be checked before submitting the form, enforcing user agreement or acknowledgment of certain terms or conditions.
